# Cosmic-Garden
A single-file browser universe where you launch planets, stars, black holes, comets, and life into a living gravity sandbox.

**https://interstellar-hitchhiker.github.io/Cosmic-Garden/**

**Design purpose:**
Its purpose is to turn physics into play, letting the user *feel* motion, force, orbit, chaos, balance, and emergence rather than merely read about them.

**Use:**
You use it by clicking, dragging, spawning, tuning gravity and time, saving universes, and watching small changes grow into beautiful cosmic systems.

**Why it is fun:**
It is fun because every action creates an unpredictable visual result: a stable orbit, a collision, a galaxy, a disaster, or a strange little living system.

**Why it is useful:**
It is useful as a playful learning toy, a visual experiment, a creative art generator, and a gentle introduction to systems thinking.

**Rationale:**
The deeper idea is that a good browser app should not just display information; it should give the user a small world with rules, consequences, surprise, and agency.
